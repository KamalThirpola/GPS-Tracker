📍 GPS Tracker

📖 Description

This is a GPS Tracker Android application built using Kotlin. The app provides real-time location tracking using Google Play Services and a foreground service. It demonstrates core Android development concepts such as location services, permissions handling, and Jetpack Compose UI.

🚀 Features

- Real-time GPS location tracking
- Foreground location service
- Built with Kotlin and Jetpack Compose
- Uses Google Play Services Location API
- Simple and user-friendly interface

🛠️ Technologies Used

- Kotlin
- Android Studio
- Jetpack Compose
- Google Play Services (FusedLocationProviderClient)
- Gradle (Kotlin DSL)

📂 Project Structure

GPS-Tracker/
│── app/
│── gradle/
│── build.gradle.kts
│── settings.gradle.kts

⚙️ How to Run the Project

1. Clone the repository:
   git clone https://github.com/KamalThirpola/GPS-Tracker.git
2. Open the project in Android Studio.
3. Sync Gradle files.
4. Run the app on an emulator or physical device.
